Bóveda Digital Privada (Norte)
Un ecosistema de comunicación y almacenamiento seguro para parejas.
Este proyecto es una Aplicación Web Full-Stack diseñada para gestionar de forma privada y segura recuerdos multimedia, notas personales y una agenda compartida. Desarrollada con un enfoque en la privacidad total y la experiencia de usuario personalizada.





🚀 Capacidades Técnicas
La aplicación integra las tecnologías más robustas del desarrollo web moderno:
Backend: Desarrollado en Python utilizando el framework Flask para una gestión eficiente de rutas y lógica de servidor.
Base de Datos: Implementación de PostgreSQL (SQL) para garantizar la persistencia de datos, perfiles de usuario y comentarios.
Almacenamiento en la Nube: Integración completa con la API de Cloudinary para el manejo, subida y optimización de imágenes en tiempo real.
Seguridad: Sistema de Autenticación Robusta con cifrado de contraseñas y control de sesiones privadas (Login/Registro).
Frontend Dinámico: Interfaz construida con HTML5 y CSS3 avanzado, incluyendo un sistema de música integrado y una introducción animada personalizada.
🛠️ Funciones Principales
Gestión Multimedia (CRUD): Capacidad para subir, visualizar, comentar y eliminar fotografías de manera instantánea.
Muro de Notas: Un espacio para el intercambio de mensajes y reflexiones con persistencia de datos.
Panel de Administración: Herramientas para formatear contenido y gestionar autorizaciones de acceso.
Reproductor de Audio: Integración de ambiente sonoro personalizado durante la navegación.
Despliegue Profesional: Configurada para integración continua (CI/CD) mediante GitHub y Render.
